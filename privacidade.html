Política de Privacidade

Sua privacidade é importante para nós. Esta política explica quais dados o Minha Rota RP coleta, como os utiliza e como os protege. Ao usar o app, você concorda com estas práticas.

1. Dados coletados

1.1 Cadastro

  • Nome completo e e-mail fornecidos no registro
  • Endereço base do representante (ponto de partida de rotas)

1.2 Localização GPS

  • Solicitada apenas quando você usa "Usar minha localização atual" para definir ponto de partida de rota
  • Coordenadas de endereços de clientes (via Google Maps e ViaCEP)

O GPS é opcional. Recusar não prejudica as demais funcionalidades.

1.3 Dados operacionais

  • Clientes: nome, CNPJ, endereço, cidade, telefone, segmento
  • Visitas: data, hora, observações, pedidos e orçamentos
  • Financeiro: receitas, comissões, gastos com clientes
  • Rotas planejadas, tarefas, lembretes e agenda

1.4 Dados técnicos

  • Logs de autenticação gerenciados pelo Supabase

2. Como usamos os dados

Usamos suas informações exclusivamente para:

  • Fazer o app funcionar corretamente
  • Autenticar seu acesso com segurança
  • Gerar relatórios e análises para uso próprio
  • Otimizar rotas via Google Maps
  • Armazenar histórico de visitas, pedidos e finanças
Seus dados nunca são vendidos, repassados ou compartilhados com terceiros para fins comerciais ou publicitários.

3. Armazenamento e segurança

Todos os dados são armazenados no Supabase (infraestrutura AWS) com:

  • Criptografia em trânsito (HTTPS/TLS)
  • Criptografia em repouso
  • Controle de acesso por Row Level Security (RLS) — cada representante acessa apenas seus próprios dados

Em caso de incidente de segurança que afete seus dados, você será notificado por e-mail.

4. Compartilhamento

Seus dados podem transitar pelos seguintes serviços:

  • Supabase Inc. — banco de dados e autenticação
  • Google LLC — geocodificação e otimização de rotas (Google Maps API)

Esses provedores operam como processadores de dados e possuem suas próprias políticas de privacidade. Nenhum outro terceiro tem acesso às suas informações.

5. Seus direitos

Você pode a qualquer momento:

  • Acessar os dados armazenados sobre você
  • Corrigir informações incorretas direto no app
  • Exportar dados via relatórios PDF disponíveis no app
  • Solicitar a exclusão completa da conta e dados
Para solicitar exclusão, envie e-mail para contato@minharotarp.com.br com o assunto "Exclusão de dados — Minha Rota RP". Prazo de atendimento: 15 dias úteis.

6. Retenção

Seus dados são mantidos enquanto a conta estiver ativa. Após exclusão, são removidos dos servidores em até 30 dias, salvo obrigação legal de retenção.

7. Atualizações

Esta política pode ser atualizada. Mudanças relevantes serão comunicadas por e-mail ou notificação no app. O uso continuado após as alterações implica aceitação.

8. Contato

  • E-mail: contato@minharotarp.com.br
